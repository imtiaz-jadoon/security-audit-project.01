## Overview
This project presents a **security audit and risk assessment** conducted for a fictional retail organization, NovaMart Retail Systems. It demonstrates practical skills expected of a **Junior Cybersecurity / SOC Analyst**.

## Scope
- Employee endpoints and internal network  
- Customer and payment data systems  
- On-premises servers and legacy systems  
- Physical security controls  
- Policies and compliance practices  

## Key Findings
- High overall risk due to excessive data access and missing controls  
- No encryption for sensitive or payment data  
- Weak password practices and no password manager  
- No disaster recovery or backup strategy  
- Limited monitoring (no IDS)

## Standards Referenced
- NIST Cybersecurity Framework (CSF)  
- PCI DSS  
- GDPR  
- SOC principles  

## Recommendations
- Enforce Least Privilege and role-based access  
- Encrypt sensitive data at rest and in transit  
- Implement backups, disaster recovery, and IDS  
- Deploy centralized password management  

## Disclaimer
This project is based on a fictional organization and was created for educational and professional portfolio purposes.
